VR Cricket Game – A-Frame Project
A fully immersive VR-based cricket game built using A-Frame. Enter a stadium, swing your bat, hear the crowd cheer, and play an arcade-style game.

Features:
1. 3D Cricket Ground with realistic bat and ball physics
2. Fully immersive VR Mode support
3. Interactive bat with collision detection and scoring
4. Increasing ball speed to raise difficulty level
5. Looping crowd cheer with reactive audio
6. Hit & Miss counter with Game Over condition
7. Works on desktop & VR headsets(Mobile supportable)

 How to Play:
1. Click Start or enter VR mode to begin the game.
2. Balls are thrown towards you — hit them by swinging the bat using spacebar key!
3. Speed increases every 5 balls. You get 5 chances.
4. Game ends after 5 misses. Highest score wins!
